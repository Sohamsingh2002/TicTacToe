
# Multiplayer Tic Tak Toe Game

This is Multiplayer TicTakToe game


## Compilation 

- Compilation
```bash
  javac *.javac
```
- Running Server
```bash
  java Server
```
- Running Client

```bash
  java Client <Enter Ip of Server>
```
## Tech Stack
- Computer Networking
- Java Socket Programming
- Java Multithreading
